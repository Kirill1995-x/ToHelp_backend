Защита данных

Администрация приложения «ВПомощь» (далее Приложение) не может передать или раскрыть информацию, предоставленную пользователем (далее Пользователь), третьим лицам. Персональные данные могут быть предоставлены:

  • специалистам для ведения их деятельности по обслуживанию пользователей;
  • в случаях, предусмотренных законодательством страны;
  • в случаях, когда администрация сочтет необходимым предоставление личной информации для соблюдения юридической процедуры или судебного распоряжения;
  • при осуществлении технической поддержки необходимой для работы Пользователя с Приложением.

В других случаях, ни при каких условиях, информация, которую Пользователь передает Приложению, не будет раскрыта третьим лицам.

Получение персональной информации

Для коммуникации в Приложении пользователь обязан ввести некоторую персональную информацию.

Использование персональной информации

Приложение использует личную информацию для обслуживания Пользователя.

Перечень обрабатываемой персональной информации: фамилия, имя, отчество, пол, дата рождения, email, номер телефона, адрес регистрации, адрес фактического места проживания, сведения об интересах и существующих проблемах, названия учебных заведений, уровень образования, профессия, семейное положение.

Ссылки

В Приложении содержатся ссылки на другие сайты. Приложение не несет ответственности за содержание, качество и политику безопасности этих сайтов. Данное соглашение о конфиденциальности относится только к информации, размещенной непосредственно в Приложении.

Безопасность

Приложение обеспечивает безопасность учетной записи Пользователя от несанкционированного доступа.

Уведомления об изменениях

Администрация Приложения оставляет за собой право вносить изменения в Политику конфиденциальности без дополнительных уведомлений. Нововведения вступают в силу с момента их опубликования. Пользователи могут отслеживать изменения в Политике конфиденциальности самостоятельно.
